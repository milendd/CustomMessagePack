# CustomMessagePack

Custom implementation of MessagePack using Elixir

## Supported types from [specification](https://github.com/msgpack/msgpack/blob/master/spec.md)
*  nil
*  true
*  false
*  uint 8
*  uint 16
*  uint 32
*  uint 64
*  int 8
*  int 16
*  int 32
*  int 64
*  fixstr
*  str 8
*  str 16
*  str 32
*  fixarray
*  array 16
*  array 32
*  negative fixint
